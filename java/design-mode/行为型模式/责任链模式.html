介绍 ​

让多个处理类处理某个请求

实际场景 ​

javax.servlet.Filter#doFilter()

优点 ​

  • 将请求与处理解耦
  • 简化处理对象,无需知道其他对象
  • 动态增删处理逻辑

缺点 ​

  • 并不一定会被执行,有可能所有处理都不适用(可以通过默认处理解决)
  • 方法嵌套过多,不容易纠错

示例场景 ​

有一个请求传入传入用户id,username值(非必须)来查询唯一用户信息

Request.java

java
@Data
@AllArgsConstructor
public class Request {
    private Integer id;
    private String username;
}
